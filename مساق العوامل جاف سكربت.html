العوامل المنطقية في جافا سكربت:

العوامل المنطقية تُستخدم لاتخاذ قرارات بناءً على القيم الحقيقية (true) أو الخاطئة (false). من العوامل الشائعة:

  • && (AND): يكون الناتج true إذا كانت جميع الشروط true.
  • || (OR): يكون الناتج true إذا تحقق شرط واحد على الأقل.
  • ! (NOT): يعكس القيمة (true تصبح false والعكس صحيح).

أمثلة:

         let isAdult = true;
         let hasPermission = false;
 
         // عامل AND
         console.log(isAdult && hasPermission); // false
 
         // عامل OR
         console.log(isAdult || hasPermission); // true
 
         // عامل NOT
         console.log(!isAdult); // false

العوامل الرياضية في جافا سكربت:

تُستخدم العوامل الرياضية لإجراء العمليات الحسابية. من أشهرها:

  • +: الجمع.
  • -: الطرح.
  • *: الضرب.
  • /: القسمة.
  • %: باقي القسمة.
  • **: الأس (الرفع إلى القوة).

أمثلة:

         let x = 10;
         let y = 3;
 
         console.log(x + y); // 13
         console.log(x - y); // 7
         console.log(x * y); // 30
         console.log(x / y); // 3.333...
         console.log(x % y); // 1
         console.log(x ** y); // 1000 (10^3)
