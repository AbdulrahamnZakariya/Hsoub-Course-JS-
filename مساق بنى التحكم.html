بُنى التحكم (Control Structures)

التفرعات الشرطية (Conditional Statements)

التفرعات الشرطية تُمكّن البرنامج من اتخاذ قرارات بناءً على شروط معينة.

مثال:

        let age = 12;
        if (age > 14) {
            console.log('Age is above 14');
        } else if (age > 10) {
            console.log('Age is above 10');
        } else {
            console.log('Age is under or equal to 10');
        }

التعبير الشرطي (Conditional Expressions)

التعبير الشرطي يُستخدم لاتخاذ قرارات متعددة بناءً على شروط مختلفة.

مثال:

    let mark = 40;
    if (mark >= 80) {
        console.log('Very Good');
    } else if (mark >= 60) {
        console.log('Good');
    } else if (mark >= 50) {
        console.log('Acceptable');
    } else {
        console.log('Failed');
    }

العامل الثلاثي (Ternary Operator)

العامل الثلاثي هو صيغة مختصرة لاستخدام if...else.

مثال:

    let mark = 40;
    let repeat = 1;
    console.log(repeat > 0 ? 'Sorry, you can\'t repeat.' : 'You can try again.');

ما هي جملة السويتش؟

جملة switch تُستخدم لاتخاذ قرارات بناءً على قيمة معينة. تُعتبر بديلاً لجملة if-else عندما يكون هناك عدة حالات.

أمثلة على السويتش

مثال 1: الأيام في الأسبوع

هذا المثال يوضح كيفية استخدام switch لمعرفة رسالة اليوم.

let day = 'Monday';

switch (day) {
case 'Monday':
    console.log('Start the week!');
    break;
case 'Friday':
    console.log('Weekend is near!');
    break;
case 'Sunday':
    console.log('Time to relax!');
    break;
default:
    console.log('It is a regular day.');
}

مثال 2: الفصول الدراسية

استخدام switch لتحديد الفصل الدراسي الحالي بناءً على الشهر.

let month = 4; // أبريل

switch (month) {
case 1:
case 2:
case 3:
    console.log('Winter Semester');
    break;
case 4:
case 5:
case 6:
    console.log('Spring Semester');
    break;
case 9:
case 10:
case 11:
    console.log('Fall Semester');
    break;
default:
    console.log('Summer Break');
}

الفرق بين switch و if-else

إليك الفرق بين switch وif-else:

  • switch: يُفضل استخدامه عند وجود قيمة واحدة يتم مقارنتها بعدة حالات.
  • if-else: أكثر مرونة للحالات المعقدة مثل استخدام عمليات المقارنة (<، >) أو العمليات المنطقية (&&، ||).

مثال باستخدام if-else:

let day = 'Monday';

if (day === 'Monday') {
console.log('Start the week!');
} else if (day === 'Friday') {
console.log('Weekend is near!');
} else if (day === 'Sunday') {
console.log('Time to relax!');
} else {
console.log('It is a regular day.');
}

حلقات التكرار (For Loop)

حلقات التكرار تُستخدم لتكرار تنفيذ جزء معين من الكود لعدد معين من المرات. يُمكنك استخدامها لتكرار العمليات حتى تتحقق شرط معين. إحدى أشهر أنواع الحلقات هي حلقة for.

هيكل حلقة for:

for (initialization; condition; increment/decrement) {
    // الكود المراد تكراره
}

فيما يلي أمثلة على استخدام حلقات for:

مثال 1: طباعة الأرقام الزوجية من 1 إلى 10

for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        console.log(i);
    }
}

مثال 2: مضاعفات رقم معين

let num = 4;
for (let i = 1; i < 10; i++) {
    console.log(num * i);
}

مثال 3: حلقة متداخلة لطباعة القيم

for (let i = 0; i < 5; i++) {
    for (let j = 1; j < 4; j++) {
        console.log(j);
    }
}

ملاحظة:

تُستخدم حلقة for بشكل أساسي عند معرفة عدد مرات التكرار المطلوب مسبقًا.
