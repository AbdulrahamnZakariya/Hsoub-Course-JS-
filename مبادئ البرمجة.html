جافا سكربت في المتصفح

تاريخ جافا سكربت

جافا سكربت تم إنشاؤها عام 1995 بواسطة Brendan Eich خلال عشرة أيام فقط، لتكون لغة برمجة تعمل على المتصفحات. أصبحت لاحقًا أحد الأعمدة الأساسية لتطوير الويب بجانب HTML وCSS.

استخدامات جافا سكربت

تُستخدم جافا سكربت في إنشاء المواقع التفاعلية، بناء تطبيقات الويب، تطوير الألعاب، وحتى في تطبيقات سطح المكتب باستخدام أطر عمل مثل Electron.

فوائد جافا سكربت

  • سهولة التعلم والاستخدام.
  • دعم واسع من جميع المتصفحات الحديثة.
  • مجتمع كبير مليء بالموارد والمكتبات.

أول كود JavaScript

كتبنا أول كود جافا سكربت وعرضنا رسالة "Hello, World!" في المتصفح. تم الربط بين ملف HTML وملف JavaScript بنجاح.

الكود:

        
        <!DOCTYPE html>
        <html>
        <head>
            <title>أول كود جافا سكربت</title>
        </head>
        <body>
            <h1>مرحبًا بالعالم!</h1>
            <script src="script.js"></script>
        </body>
        </html>

        // ملف script.js
        console.log('Hello, World!');

جافا سكربت في المتصفح

ما تعلمته في جافا سكربت

الفرق بين var و let:

        let name = 'Mohammad';
        let age = 16;
        console.log(name);
        console.log(age);

أساسيات كتابة المتغيرات في جافا سكربت:

  • Must be meaningful.
  • Cannot start with a number.
  • Cannot contain spaces or dashes.
  • Case sensitive.
  • Cannot use reserved keywords (e.g., var, let, if, else).

أمثلة على المتغيرات:

        let name = 'Ahmad'; // String "", ''
        let age = 20; // Number
        let isStudent = true; // Boolean true, false
        let grade; // Undefined
        let empty = null; // Null

الفرق بين null و undefined:

Null تعني أن القيمة فارغة أو غير معروفة، بينما undefined تعني أن المتغير تم تعريفه ولكن لم يتم إعطاؤه أي قيمة.

الفرق بين اللغات الثابتة والديناميكية:

        let isStudent = true;
        console.log(typeof isStudent); // "boolean"
        isStudent = 10;
        console.log(typeof isStudent); // "number"

عن المصفوفات:

        let students = ['Ahmad', 'Kamal'];
        console.log(students);
        students[2] = 10;
        console.log(students);
